Detailed Description

UCX backend based on UCP STREAM.

UCP STREAM provides a byte-stream semantics (TCP-like), therefore MTCL implements message framing in userspace (header + payload).

It is important not to desynchronize the stream when multiple asynch receives are posted concurrently.
