void MTCL::Handle::close ( bool  close_wr = true,
bool  close_rd = true 
)
inlinevirtual

Close the handle for writing and/or reading.

close_wr disables further sends. close_rd disables further receives. After closing, the runtime may release resources associated with this handle.

Parameters
close_wrIf true, close the write side (no further sends).
close_rdIf true, close the read side (no further receives).

Implements MTCL::CommunicationHandle.

void MTCL::Handle::yield ( )
inlinevirtual

Yield the handle back to the runtime.

This function informs the runtime that the handle is no longer held for user-side progress, so the runtime may resume managing receive-side progress for this handle.
