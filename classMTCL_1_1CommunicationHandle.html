virtual ssize_t MTCL::CommunicationHandle::probe ( size_t &  size,
const bool  blocking = true 
)
pure virtual

Check for incoming message and write in size the amount of data present in the message.

Parameters
[out]sizetotal size in byte of incoming message
[in]blockingif true, the probe call blocks until a message is ready to be received. If false, the call returns immediately and sets errno to EWOULDBLOCK if no message is present on this handle.
Returns
ssize_t sizeof(size_t) upon success. If -1 is returned, the error can be checked via errno.
